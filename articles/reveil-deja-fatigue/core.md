## Angle éditorial

Se réveiller épuisé (ou en hyperéveil à 4h) n'est pas un problème de caractère. C'est une désynchronisation entre trois systèmes biologiques qui pilotent normalement le réveil en coopération : axe HPA (cortisol), système sympathique (noradrénaline/dopamine), horloge centrale (SNC). L'article identifie deux profils opposés issus du même mécanisme, et propose des leviers pratiques.

---

## Points clés (résumé éditorial)

1. **La chorégraphie du réveil normal** : cortisol (CAR) + noradrénaline (tonus sympathique) + dopamine (initiation motrice) — trois systèmes couplés.

2. **Désynchronisation des 3 horloges** : dans le Covid long / EM-SFC / fibromyalgie, le couplage cortisol ↔ sympathique ↔ horloge centrale se défait.  
   - Profil A (hyperéveil) : réveil 4–6h, adrénaline, palpitations  
   - Profil B (démarrage impossible) : yeux ouverts, corps présent, pas d'impulsion

3. **Note auteur clinique** : Rémy a vécu le profil A — anxiété du corps sans anxiété mentale, atténuée par hydroxyzine (antihistaminique central) → confirmation indirecte d'une composante histaminergique dans la décharge sympathique nocturne.

4. **Intolérance orthostatique** : noradrénaline = vasoconstricteur périphérique. Tonus sympathique bas → passage couché→debout difficile. Conséquence hémodynamique directe de l'axe 2.

5. **Dette ATP nocturne** : alpha-delta intrusion (fibromyalgie), fragmentation du sommeil (Covid long) → restauration cellulaire insuffisante même avec une nuit "complète".

6. **Leviers pratiques** : lever progressif, hydratation, fractionnement du matin, respiration lente, régularité circadienne, observation/journaling (→ Boussole).
